## Quick Start

1. **Local Testing**: Open `index.html` in your browser
2. **Deploy to GitHub**:
   ```bash
   git add .
   git commit -m "Initial website launch"
   git push origin main
   ```
3. **Enable GitHub Pages**: 
   - Settings → Pages → Select main branch → Save
   - Live at: `https://low-tide-fragrances.github.io/shop`
